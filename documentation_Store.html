Документация проекта "Store"

1. Описание проекта

Этот проект представляет собой интернет-магазин, который предоставляет пользователям возможность покупать товары через корзину. Пользователи могут авторизоваться через стандартную форму или с помощью социальных аккаунтов (через GitHub).

Основные возможности:

  1. Категории продуктов.
  2. Просмотр, добавление и удаление продуктов из корзины.
  3. Регистрация и авторизация пользователей.
  4. Интеграция с GitHub для аутентификации через OAuth.

Технологии

  1. Django 3.2.12
  2. PostgreSQL
  3. HTML/CSS/JS для фронтенда
  4. Allauth для аутентификации
  5. GitHub OAuth для входа через сторонние сервисы

2. Установка

Требования

  1. Python 3.12+
  2. Django 5.1.6+

Шаги по установке

  1. Клонируйте репозиторий: 
    git clone <repository_url>
cd store
  2. Создайте и активируйте виртуальное окружение: 
    python -m venv venv
source venv/bin/activate  # Для Linux/MacOS
.\venv\Scripts\activate  # Для Windows
  3. Установите зависимости: 
    pip install -r requirements.txt
  4. Установите PostgreSQL: 
    sudo apt-get install postgresql
  5. Создайте базу данных и пользователя, затем настройте параметры подключения в файле settings.py: 
    DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql_psycopg2',
        'NAME': 'store_db', # заменить имя бд
        'USER': 'store_username', # заменить на имя пользователя для бд
        'PASSWORD': 'store', # заменить на пароль к бд
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
  6. Работа с базой данных: 
    python manage.py makemigrations
python manage.py migrate
  7. Настройка фикстур: 
    python manage.py loaddata store/products/fixtures/categories.json store/products/fixtures/goods.json
  8. Создайте суперпользователя: 
    python manage.py createsuperuser
  9. OAuth с GitHub: 
    SOCIALACCOUNT_PROVIDERS = {
    'github': {
        'SCOPE': ['user', 'repo', 'read:org'],
        'APP': {
            'client_id': 'clien_id',  # Заменить на реальный client_id
            'secret': 'secret_key',  # Заменить на реальный secret_key
            'key': 'github',
        }
    }
}
  10. Запустите сервер разработки: 
    python manage.py runserver
  11. Сайт будет доступен по адресу: [http://localhost:8000](http://localhost:8000)

3. Аутентификация

Проект поддерживает как стандартную аутентификацию через форму, так и аутентификацию через социальные сети (GitHub). Вся аутентификация на сайте обеспечивается библиотекой django-allauth.

4. Планируемые улучшения

  1. Рассылка на email — планируется добавить функциональность отправки писем для подтверждения почты.
  2. Тесты — проекту требуется покрытие тестами для повышения надежности.
  3. Кеширование — добавление кеширования для улучшения производительности.
  4. Фоновые задачи — планируется внедрение Celery для обработки задач в фоновом режиме.
  5. Страница заказов — необходимо реализовать отдельную страницу для управления заказами пользователей
  6. Исправление подсчета суммы в профиле — на данный момент сумма увеличивается только при многократном добавлении товаров из каталога. Необходимо реализовать корректный пересчет внутри профиля.

Заключение

Этот проект на базе Django позволяет пользователям просматривать товары, управлять корзинами и выполнять другие типичные действия для интернет-магазина. Социальная аутентификация через GitHub и кастомные модели пользователей обеспечивают безопасность и настраиваемость.
